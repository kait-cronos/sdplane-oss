* Ubuntu 22.04.4 LTS (Server) を OS install
  - Ubuntu 20.04.6 のインストーラーUSBを使用。
  - English を選ぶ。
  - Installer update available で continue without updating.
  - keyboard layout: English / English で Done.
  - DHCP アドレス取れてることを確認して Done.
  - Proxy 空白で Done.
  - Mirror: jp.archive で Done.
  -> 一瞬 crash reboot って見える。
  -> Sorry, installer has restarted because of an error. で終わった。

* timezone 設定
  # timedatectl set-timezone Asia/Tokyo

* apt install

  (以下は全部 optional だが、install しておくと良い)
  - git, tig, wget
  - etckeeper
  - iptables, iptables-persistent
  - fail2ban
  - dmidecode
  - ripgrep
  - screen, tmux

* https://github.com/halayli/lthread を compile & install
  <https://lthread.readthedocs.io/en/latest/intro.html#installation>

* DPDK prerequisites:
  - apt install python3
  - pip3 install meson ninja
  - apt install python3-pyelftools
  - apt install libnuma-dev

* DPDK 23.11 を install (試したのは23.11.0だったが、大差ないはず)
  <https://fast.dpdk.org/rel/dpdk-23.11.1.tar.xz>
  # tar xJf dpdk-<version>.tar.xz
  # cd dpdk-<version>
  # meson setup build
  # cd build
  # ninja
  # meson install
  # ldconfig

  確認:
  yasu@partaker:~$ pkg-config --modversion libdpdk
  23.11.0

* Hugepage メモリ設定
  (2MB hugepage を1500, 1GbE x 2 で必要。NICによってもっと必要かも。)
  # vi /etc/default/grub:
    GRUB_CMDLINE_LINUX="hugepages=1536"
  # update-grub
